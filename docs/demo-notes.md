# Demo Notes

Use this script when recording the short assignment walkthrough:

1. Start MongoDB, backend, and frontend.
2. Login with the seeded admin account.
3. Open the dashboard and show the stats cards.
4. Create a visitor record.
5. Create an appointment for that visitor.
6. Approve the appointment from the queue.
7. Issue a pass and show the generated QR image.
8. Verify the pass code in the scanner panel.
9. Trigger check-in and check-out.
10. Export the logs CSV.
